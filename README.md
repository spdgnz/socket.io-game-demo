# Socket.io Agar.io Game Clone
Questo progetto è ispirato al gioco su Browser agar.io, il gioco permette a piu giocatori di collegarsi sulla stessa mappa.
Il gioco consiste nello spostarsi sopra le Orb generarte in punti casuali dell'area di gioco per aumentare il proprio punteggio.
Inoltre è possibile spostarsi su un altro giocatore con un punteggio inferiore al proprio per eliminarlo.

## Avvio della demo
Step per l'avvio della demo:

Navigare alla root del progetto.

Eseguire il comando per scaricare le dipendenze necessarie.
```powershell
npm install
```
Eseguire il comando per l'avvio del server.
```powershell
node .\server\index.js
```
Da Browser navigare al seguente link <http://localhost:7000><br>
Aprire piu pagine contemporaneamente per simulare lo scambio di messaggi fra più utenti.

Se i comandi non dovessero funzionare sarà necessario scaricare Node al seguente [link](https://nodejs.org/en/download)
